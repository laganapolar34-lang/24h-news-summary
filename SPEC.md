# 新闻简报网页 - 技术规格文档

## 1. 项目概述

- **项目名称**：24小时新闻简报
- **项目类型**：静态新闻聚合网页
- **核心功能**：每日自动更新的国际新闻、黄金价格、汇率展示
- **目标用户**：需要快速了解当日国际要闻的用户

## 2. 技术架构

### 2.1 目录结构
```
24h-news-summary/
├── index.html          # 首页（当天最新）
├── YYYY-MM-DD.html    # 历史页面（自动生成）
├── SPEC.md            # 本规格文档
├── README.md          # 项目说明
├── netlify.toml       # Netlify 配置
├── css/
│   └── style.css      # 样式文件
└── js/
    └── main.js        # 交互脚本
```

### 2.2 技术栈
- **前端**：HTML5 + CSS3 + Vanilla JavaScript
- **托管**：GitHub Pages (Netlify)
- **更新**：每日 Cron 任务自动生成

### 2.3 核心模块
| 模块 | 职责 |
|------|------|
| Header | 标题、日期导航、日期选择器 |
| GoldSection | 黄金价格、汇率展示 |
| NewsSection | 新闻分类展示 |
| DateNavigation | 前一天/后一天切换 |

## 3. UI/UX 规格

### 3.1 布局
- 单列移动端优先设计
- 最大宽度 800px
- 居中显示

### 3.2 色彩方案
- 主色：#1d1d1f（文字）
- 次色：#86868b（辅助文字）
- 强调色：#0066cc（链接）
- 背景：#ffffff
- 卡片背景：#f5f5f7

### 3.3 字体
- 标题：-apple-system, SF Pro Display
- 正文：SF Pro Text
- 字号：标题 48px，正文 17px

### 3.4 组件
- 导航按钮：圆角 20px，背景 #f5f5f7
- 黄金卡片：圆角 16px，轻阴影
- 新闻卡片：圆角 20px，悬浮动画

## 4. 功能规格

### 4.1 日期导航
- 前一天/后一天按钮
- 日期选择器（限制在今天及之前）
- 首页显示"今日"标签
- 历史页面显示具体日期

### 4.2 数据展示
- 黄金价格：纽约现货、伦敦现货、人民币/克
- 汇率：USD/CNY、100JPY/CNY
- 新闻：政治、财经、军事分类

### 4.3 响应式
- 移动端优先
- iPhone 安全区域适配
