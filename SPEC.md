# 24小时新闻摘要 - 项目规格说明书

## 1. 项目概述

- **项目名称**: 24小时新闻摘要
- **项目类型**: 新闻聚合网页应用
- **核心功能**: 聚合政治、财经、军事新闻，实时获取黄金价格，生成每日新闻摘要网页
- **目标用户**: 需要快速浏览每日要闻的用户

## 2. 功能需求

### 2.1 新闻收集
- 抓取最近24小时的新闻
- 分类：政治、财经、军事
- 每个新闻包含：标题、摘要、来源、发布时间、原文链接

### 2.2 黄金价格
- 实时获取纽约金价（美元/盎司）
- 实时获取伦敦金价（美元/盎司）
- 自动转换为人民币/克（汇率按当日汇率）

### 2.3 网页展示
- 24小时新闻摘要总览
- 黄金价格看板（美元/盎司 + 人民币/克）
- 新闻分类展示（政治、财经、军事）
- 点击标题可跳转到原文

### 2.4 数据存储
- 每日新闻自动保存为JSON文件
- 支持历史新闻查询
- 数据存储在本地文件系统

## 3. 技术方案

### 3.1 架构
- **前端**: 单页HTML应用，响应式设计
- **后端**: Python Flask API
- **数据存储**: 本地JSON文件
- **定时任务**: 每日自动更新

### 3.3 API端点
- `GET /api/news` - 获取当日新闻
- `GET /api/news?date=YYYY-MM-DD` - 获取指定日期新闻
- `GET /api/gold` - 获取黄金价格
- `GET /api/dates` - 获取有数据的日期列表
